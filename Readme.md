# Zomato Restaurants Data Analysis 🍴

This project provides a comprehensive analysis of Zomato restaurant data to uncover insights into market trends, pricing strategies, and customer satisfaction. The analysis is aimed at helping restaurateurs identify "sweet spots" for pricing and service offerings.

## 📌 Project Overview
The goal of this analysis was to understand the factors that contribute to high restaurant ratings and to identify market gaps. [cite_start]The project involves data cleaning, exploratory data analysis (EDA), and data visualization using Python. [cite: 1, 3]

## 📊 Dataset Description
[cite_start]The dataset contains information on over 211,000 restaurant listings[cite: 1, 3]. Key columns include:
- **res_id**: Unique identifier for each restaurant.
- **name & establishment**: Restaurant name and type.
- **city & locality**: Geographic location details.
- **cuisines**: Types of food offered.
- **average_cost_for_two**: The typical cost for a meal for two.
- **aggregate_rating**: Customer rating on a scale of 0-5.
- [cite_start]**votes**: Number of customer reviews. [cite: 1, 3]

## 🛠️ Tech Stack
- **Language:** Python
- [cite_start]**Libraries:** Pandas, NumPy, Matplotlib, Seaborn, SciPy 
- **Tool:** Jupyter Notebook

## 🔍 Key Insights & Findings
- [cite_start]**The "Mass Market" Trap:** Restaurants in the ₹201 - ₹500 range are the most competitive (44% of market share) but often suffer from mediocre ratings (average 3.39). 
- [cite_start]**The Value-Premium Sweet Spot:** A significant jump in ratings (from 3.59 to 3.81) is observed in the ₹801 - ₹1,200 range. [cite: 1, 3]
- [cite_start]**Peak Satisfaction:** The ₹1,201 - ₹2,000 price bracket yields the highest average rating (4.09), representing the segment where customers are most satisfied. [cite: 1, 3]
- [cite_start]**Recommendation:** For new restaurateurs, targeting the ₹1,200 - ₹1,500 range offers a balance of high perceived quality and lower competition than the mass market. [cite: 1, 3]
